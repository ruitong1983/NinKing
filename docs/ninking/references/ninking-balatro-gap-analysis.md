# NinKing vs Balatro — 差距分析与优化建议

> 基准：`11-ninja-cards.md`（45 张忍者牌） vs `balatro-joker-design.md`（150 张小丑牌）
> 日期：2026-06-14

---

## 一、总览：结构差距

| 维度 | Balatro | NinKing | 差距判断 |
|------|---------|---------|---------|
| 总卡牌数 | 150 | 45 | 体量差距合理（项目阶段不同） |
| 普通占比 | ~50% (75) | 20% (9) | **严重偏低** — 普通卡是商店主体 |
| 罕见占比 | ~37% (55) | 38% (17) | 比例合理 |
| 稀有占比 | ~10% (15) | 31% (14) | **严重偏高** — 稀有过多稀释稀缺感 |
| 传奇占比 | ~3% (5) | ~7% (3) | 略高但可接受 |
| 效果维度数 | 10 | 10 | 数量持平，但各维度深度差异大 |

**核心问题：稀有度倒挂。** NinKing 普通卡只有 9 张，商店刷新池太浅，玩家会反复看到同样卡牌。

## 二、效果维度逐一对比

### 2.1 筹码类（+Chips）

| | Balatro | NinKing |
|------|---------|---------|
| 数量 | ~20 张 | 6 张 (通用+组别定向各含 chips) |
| 触发方式 | 牌型/花色/点数/手牌/剩余弃牌/每回合 | 几乎全是无条件或组别条件 |
| 成长型 | 跑步选手(顺子+15永久)、方形小丑(4张+4永久)、城堡(弃牌+3) | 无纯 chips 成长型 |
| 衰减型 | 冰淇淋(+100, 每出牌-5) | 无 |

**缺失：** 纯 chips 成长型、衰减型(风险回报)、手牌数量→chips、剩余资源→chips。

### 2.2 倍率类（+Mult）

| | Balatro | NinKing |
|------|---------|---------|
| 数量 | ~35 张 | ~10 张（分布多个子类） |
| 触发方式 | 花色/点数/牌型/斐波那契数列/每张小丑/售价 | 组别条件为主 |
| 概率触发 | 开心小丑(对子+8)、印错小丑(随机0~23) | 无 |
| 回合衰减 | 爆米花(+20, -4/回合)、香蕉(1/6自毁) | 无 |

**缺失：** 概率触发型、回合衰减型、数列主题触发(斐波那契)、相邻卡牌联动。

### 2.3 倍乘类（×Mult）

| | Balatro | NinKing |
|------|---------|---------|
| 数量 | ~30 张 | ~10 张（分布多个子类） |
| 成长倍乘 | 星座(×0.1/星球)、钢铁(×0.2/钢铁牌)、招财猫(×0.25/幸运)、全息(×0.25/加牌)、篝火(×0.25/售出) | 仅成长型 +mult，无成长型 ×mult |
| 条件重置 | 方尖石塔(打非最常用牌型,重置) | 无（头悬梁/尾刺骨已重做） |
| 手中牌触发 | 男爵(K×1.5)、射月(Q+13) | 影之眷顾(J/Q/K+3)、王牌侍从(A+5) |

**缺失最严重：** 成长型倍乘完全空白。这是 Balatro 后期构筑的核心——星座、钢铁小丑、全息影像等提供缓慢但不可逆的 ×mult 成长。

### 2.4 经济类

| | Balatro | NinKing |
|------|---------|---------|
| 数量 | ~17 张 | 6 张 |
| 回合被动收入 | 黄金小丑($4/回合)、火箭($1+递增)、卫星($1/星球种类) | 无回合被动收入 |
| 弃牌产钱 | 无面小丑(弃3人头$5)、邮寄回扣(弃对应点数$5) | 无 |
| 计分产钱 | 名片(人头1/2→$2)、璞玉(方片→$1)、黄金门票(黄金牌→$4) | 福神(喜→$2)、金尾(镀金→$3) |
| 售价成长 | 鸡蛋(自身售价+$3/回合)、礼品卡(所有+$1) | 无 |
| 利息相关 | 冲向月球(每$5+$1利息) | 利息之印(上限+$5) |

**缺失：** 回合被动收入、弃牌触发产钱、计分产钱概率类、售价成长类。

### 2.5 牌组操控 / 生成类

Balatro 有完整的牌组改造体系（加牌/删牌/复制/转换/生成消耗牌）。**NinKing 此维度完全空白。** 忍者牌组 15 张固定，目前没有任何卡牌可改变牌组构成或生成额外资源。

**机会：**
- 忍术卷轴（生成一次性增益）
- 暗杀术（移除手牌中一张弱牌 → 永久强化牌组）
- 通灵之术（随机生成一张忍者牌加入商店）

### 2.6 触发/重复类（Retrigger）

Balatro 有哑剧演员、黄昏、喜与悲、苏打水等额外触发牌。**NinKing 此维度完全空白。**

**机会：**
- 影分身之术：影组计分牌额外触发 1 次
- 时遁之术：滅组牌型判定时，重复上一次滅的牌型效果

### 2.7 售出/退役效果

Balatro 有隐形小丑(售出→复制)、摔跤手(售出→消除boss)、零糖可乐(售出→双倍标签)。**NinKing 所有忍者牌无售出效果。**

**机会：**
- 牺牲之印：售出时永久 +1 mult 给相邻忍者牌
- 临终遗言：售出时获得 $10
- 破釜沉舟：售出时当前盲注 ×2 mult

### 2.8 回合/手牌衰减型（临时强力）

Balatro 有冰淇淋(每手-5)、爆米花(每回合-4)、黑龟豆(每回合-1手牌上限)等"临时强力→逐渐衰减"设计。**NinKing 完全缺失此类型。**

这类设计的好处：
- 降低商店决策门槛（买不买？还剩几回合价值？）
- 创造"换牌窗口"策略深度
- 短期爆发 vs 长期投资取舍

## 三、设计深度差距

### 3.1 成长型系统单调

NinKing 的 5 张成长卡全是 `+N` 累积型，触发条件单一：

| NinKing 成长卡 | 触发 |
|------|------|
| 修行者 | on_play +1 mult |

Balatro 的成长体系：
- **×mult 成长**：星座(星球牌)、钢铁小丑(钢铁牌)、全息影像(加牌)、篝火(售出牌)、玻璃小丑(摧毁玻璃牌)
- **多维度触发**：出牌(超新星)、弃牌(城堡)、跳过盲注(红牌)、使用消耗牌(占卜师)、商店重掷(抽认卡)、售出(篝火)
- **衰减型**：拉面(×2 每弃1张-0.01)

### 3.2 条件维度太少

NinKing 触发条件集中在"三组牌型"，而 Balatro 覆盖：
- 花色（4 种 + 黑/红二分）
- 点数（A/2/3/.../K，奇数/偶数，人头/非人头，斐波那契数列）
- 手牌状态（剩余弃牌、手牌张数、手中特定牌）
- 经济状态（持有金钱、售价总额）
- 牌组状态（牌组张数、增强牌数量、空槽位）
- 回合进度（首回合、第N次出牌、boss关）
- 商店行为（重掷、跳过、购买）

### 3.3 缺少 Boss 互动

NinKing 有 blinds and bosses 系统（13-blinds-and-bosses.md），但**忍者牌与 Boss 无任何交互**。Balatro 有斗牛士（boss触发→$8）、摔跤手（售出消除boss）、希科（所有boss效果消失）。

## 四、优化建议 — 按优先级排序

### P0 · 基础补全（立即见效）

#### 4.1 扩充普通卡池 9→20

当前商店主体卡池太浅。新增方向（都是简单效果、低售价）：

| 建议卡牌 | 效果 | 设计参考 |
|------|------|------|
| 手裏剣·改 | +8 chips × 出牌张数 | 蓝色小丑(牌组→+2) |
| 修行短刀 | 顺子→+30 chips | 狡猾小丑(顺子+80) |
| 残影 | 剩余换牌次数×+5 chips | 旗帜(弃牌→+30) |
| 盗听 | 弃牌中每张人头牌+3 chips | — |
| 影足轻 | 偶数点数牌计分→+3 mult | 偶数史蒂文(+4) |
| 火遁·初 | 每回合首次出牌+10 mult | — |
| 风遁·初 | 每持有$3→+1 mult(上限+8) | 提靴带($5→+2) |
| 飞镖 | 1/3 概率 +15 mult | 印错小丑(随机0~23) |
| 替身木 | +8 mult, 每回合-2 | 爆米花(+20,-4) |
| 寒冰手里剑 | +80 chips, 每次出牌-10 | 冰淇淋(+100,-5) |
| 利息·改 | 回合结束$2 | 黄金小丑($4) |

#### 4.2 将部分史诗降级为稀有

当前 14 张史诗占比过高。以下可降为稀有（效果不变）：
- 铁索连环（×1.5 两组牌型相同，对标 Balatro 二重奏 ×2）
- 王牌侍从（+5 mult/Ace 上限+20，对标 Balatro 射月 Q→+13）
- 清一色（×2 三清触发，对标 Balatro 花盆 ×3）

降级后史诗 11 张，稀有 20 张，比例更健康。

#### 4.3 成长型 ×mult 卡牌（2-3 张）

这是最大的设计空白。建议：

| 建议卡牌 | 品质 | 效果 | 参考 |
|------|------|------|------|
| 星见 | 稀有 | 每使用一张秘仪卡 ×mult +0.1（起始 ×1） | 星座(星球→×0.1) |
| 试炼之印 | 史诗 | 每通过一个 Boss 盲注 ×mult +0.25 | 回溯(跳过盲注→×0.25) |
| 涅槃 | 史诗 | 影组从散牌升级到任意牌型时永久 ×mult +0.3，影组牌型降级则重置 | 方尖石塔(打非最常用→×0.2) |

### P1 · 机制扩展（丰富策略维度）

#### 4.4 衰减型卡牌（3-4 张）

"买来爆发几轮后抛弃"是 Balatro 的核心商店决策维度：

| 建议卡牌 | 品质 | 效果 |
|------|------|------|
| 雷遁之术 | 普通 | +40 mult, 每出牌-8 |
| 冰封之印 | 稀有 | ×2 mult, 每回合-×0.2 |
| 燃烧查克拉 | 稀有 | +100 chips +20 mult, 每出牌-10 chips -2 mult |
| 影打 | 普通 | +30 chips, 每弃牌-10 chips |

#### 4.5 售出/退役效果（3-4 张）

| 建议卡牌 | 品质 | 效果 |
|------|------|------|
| 牺牲之术 | 稀有 | 售出→相邻忍者牌永久 +3 mult |
| 禁术·尸鬼封尽 | 史诗 | 售出→当前盲注 ×3 mult，跳过下回合出牌 |
| 传功长老 | 稀有 | 售出→随机一张忍者牌售价 -$5 |
| 金蝉脱壳 | 普通 | 售出→获得 $8 |

#### 4.6 重复触发（Retrigger）（2-3 张）

| 建议卡牌 | 品质 | 效果 |
|------|------|------|
| 影分身 | 稀有 | 影组计分牌额外触发 1 次 |
| 刻之印 | 史诗 | 每次出牌的首张计分牌额外触发 1 次 |
| 连锁爆击 | 稀有 | 滅组含增强牌时，每张增强牌 1/3 概率额外触发 |

#### 4.7 Boss 互动卡（2-3 张）

| 建议卡牌 | 品质 | 效果 |
|------|------|------|
| 弱点看破 | 稀有 | Boss 能力触发时 +$8 | 
| 忍法·破 | 史诗 | 售出→消除当前 Boss 盲注效果 |
| 影之盾 | 稀有 | Boss 限制出牌次数 -1 → 无效化 |

#### 4.8 牌组操控（2-3 张）

| 建议卡牌 | 品质 | 效果 |
|------|------|------|
| 卷物·改 | 稀有 | 选盲注→手牌中 1 张随机获得镀金增强 |
| 暗杀术 | 稀有 | 弃 1 张→永久从牌组移除 + 获得 $5 |
| 影缝之术 | 史诗 | 出牌→手牌中 1 张随机变为万能花色 |

### P2 · 深度打磨（差异化竞争力）

#### 4.9 "喜"系统深度扩展

当前"喜"是 NinKing 的独特系统，但只有 6 张卡与之联动。建议：

| 建议卡牌 | 效果 |
|------|------|
| 喜之共鸣 | 每触发 1 个喜 → 下一次出牌 +1 出牌次数(每回合限 1 次) |
| 喜之波纹 | 喜的持续时间 +1 回合 |
| 悲之反転 | 未触发任何喜 → +15 mult（反向激励） |
| 喜之伝染 | 触发喜时 1/3 概率额外触发一个随机喜 |

#### 4.10 三组互动的更多维度

当前三组（影/瞬/滅）的互动偏重牌型比较，可扩展：

- **组间传递**：影的溢出 mult → 加到滅的 chips（或反之）
- **组牺牲**：放弃一组出牌 → 另两组获得补偿
- **组轮转**：每回合三组"C 位"轮换，C 位组获得额外倍率

具体卡牌建议：

| 建议卡牌 | 效果 |
|------|------|
| 影渡 | 影组不计分 → 影组 chips 值 ×2 加到滅组 |
| 轮转之印 | 每回合随机一组获得 ×1.5，另一组 ×0.5 |
| 舍身 | 跳过瞬组出牌 → 影和滅各 +20 mult |

#### 4.11 手牌/弃牌资源操控

| 建议卡牌 | 效果 |
|------|------|
| 忍法·空蝉 | +1 手牌上限 |
| 风魔之遁 | +2 换牌次数, -1 手牌上限 |
| 时间之印 | 每剩余 1 次出牌 → +5 mult |

## 七、NinKing 已有差异化优势（应保持）

1. **三组排列系统**：比 Balatro 的单一牌型判定更深，2D 决策空间
2. **"喜"全局系统**：Balatro 没有的独特全局状态机制，差异化核心
3. **规则变更互斥组**：`mutex_group` 强制取舍，比 Balatro 更结构化
4. **经济→数值转化曲线**：金剛力/黄金律的钱→战斗力转化是 NinKing 原创模式
5. **条件重置成长**：已移除（头悬梁/尾刺骨重做为组别定向×4/×5），当前成长线仅剩修行者的简单累积
