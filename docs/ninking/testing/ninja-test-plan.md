> **用途:** 覆盖所有计分类忍者牌的得分测试用例

> **Review:** `tools/review_test_data.py` — 每日审查Delta正确性
> **方法论:** `automated-formula-testing.md` — 测试流程/模板/检查清单
> **可视化:** `ninja-test-viewer.html`（双击打开）
> **更新:** 2026-06-14 — v5.0 列加法化

## 测试架构

每张卡分 **基础测试** + **扩展测试** 两类：

| 测试号 | 维度 | 说明 |
|:-----:|------|------|
| T1 | 主要激活 | 触发条件的最典型场景，低牌型（避喜） |
| T2 | 边界/补充 | 触发条件的另一种路径 + 中等牌型 |
| T3 | 不触发基线 | 条件不满足时的得分为对比基准 |
| **T4** | **新触发路径** | **不同于T1/T2的触发组合**（如散→同花、不同牌型） |
| **T5** | **高强度/高列乘** | **豹/同花顺混搭 + 列×64~×32768** |
| **T6** | **喜联动/极值** | **忍者效果在全同花/全黑/四张等喜下叠加** |

### 计分公式（v5.0 列加法化）

```
group_score = (card_chips + hand_chips + ench_chips + ninja_chips) 
            × (hand_mult + ench_mult + ninja_mult) 
            × ∏(ninja_x_mult) × ∏(card_x_mult)

col_score   = 同 group_score 公式，应用于纵向列（散牌列=0）
              unconditional 忍者效果同时应用于行列
              组条件(group)忍者效果不应用于列

total_raw   = Σ(head + mid + tail) + Σ(col0 + col1 + col2)

final_score = max(total_raw, 1)
            × ∏(global_xi_x_mult)         ← 全局喜×mult
```

> **v4.0→v5.0 核心变化：** 列从全局 ×mult(×2/×4/×8/×16/×32) 改为独立 chips×mult 加法项，与行平级。
> 组级喜(三清×2/三顺清×3)同时作用于行和列。

### 扩展测试设计原则

1. **T4 差异化触发路径** — 每个条件型卡用不同牌型组合触发（如同花代替顺子、豹子代替对子），验证条件检测不绑定特定牌型
2. **T5 高强度/高列分验证** — 用豹子(5)/同花顺(4) 等顶级牌型，或者三列豹+全三条×4 的高列分环境，确保公式在极端值下不溢出/不归零
3. **T6 喜系统叠加** — 在全同花×3/全黑×2/四张×5 等喜环境下验证忍者效果，确保组级喜（三清）同时影响行和列、全局喜正确应用
4. **全♠ 牌组的特殊处理** — T4/T6 中多处使用 9张全♠，会同时触发「全同花×3 + 全黑×2 + 三清×2」三重喜，是高压力测试

## 测试框架架构

```
calc(cs_h, cs_m, cs_t, ne, xi_bonus, xi_override)
  ├── eval_group()       → 3卡牌型判定（0-5）
  ├── eval_col()         → 列牌型判定
  ├── detect_xi()        → 9种喜模式检测
  ├── gs()               → 分组计分(chips×mult×x_stack)
  ├── 列独立计分          → unconditional 忍者效果同样作用于列
  ├── 组级喜应用          → 三清/三顺清/顺清打头 ×(基数+xi_bonus)，行和列均应用
  ├── total_raw 加法合成  → head + mid + tail + col_total
  └── 全局喜×mult         → 全局累乘(基数+xi_bonus)
```

### 生成器 `tools/gen_test_v2.py`

| 函数 | 作用 |
|------|------|
| `ne_all(c,m,x)` | 三组相同 ne |
| `ne_partial(**kwargs)` | 只指定变化的组，缺省归零 |
| `tc(cid,..., ne, xi_bonus, xi_override)` | 生成 baseline + with_ninja 两行 |

### 扩展说明

- **xi_bonus:** 喜鹊效果，所有喜×mult +1（组级和全局均加算）
- **xi_override:** 清一色效果，替换三清×2→×3
- **龙之眼:** 通过 ne_g['x'] 传入 per-group ×stack，calc() 累乘到组内
- **成长卡:** 基础 effect=0，通过 ne 传入假设累积值
- **三列豹牌组** `[♠3♥7♦J][♦3♠7♥J][♣3♣7♠J]` — 9张3个点数各3张，三列都是豹子(~1000分/列) + 全三条×4
