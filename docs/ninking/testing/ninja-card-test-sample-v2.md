## 一、CSV 字段定义（47列）

每行一个完整测试用例（不含忍者时基准行会单独标记 version="baseline"）。

| # | 列名 | 类型 | 说明 |
|---|------|------|------|
| 1 | `version` | str | `baseline`（无忍基准）或 `with_ninja`（带目标忍者） |
| 2 | `card_id` | str | 忍者 ID |
| 3 | `card_name` | str | 忍者名称 |
| 4 | `test_no` | str | T1/T2/T3 |
| 5 | `test_desc` | str | 测试说明 |
| **牌组布局** |
| 6-8 | `h1,h2,h3` | str | 头组 3 张牌 |
| 9-11 | `m1,m2,m3` | str | 中组 3 张牌 |
| 12-14 | `t1,t2,t3` | str | 尾组 3 张牌 |
| **牌型判定** |
| 15-17 | `h_type,m_type,t_type` | str | 牌型中文名 |
| 18-20 | `h_tc,m_tc,t_tc` | int | 牌型码(0-5) |
| 21-23 | `c0_type,c1_type,c2_type` | str | 列牌型 |
| 24 | `col_x_str` | str | 列×mult 数组字符串 e.g. "4,16,4" |
| 25 | `xi_str` | str | 喜触发列表 e.g. "三清" |
| 26 | `triggered` | bool | 该忍者条件是否触发 |
| **忍者贡献** |
| 27-29 | `nc_h_c,nc_h_m,nc_h_x` | int | 忍者对头组 chips/mult/x_mult 贡献 |
| 30-32 | `nc_m_c,nc_m_m,nc_m_x` | int | 忍者对中组 |
| 33-35 | `nc_t_c,nc_t_m,nc_t_x` | int | 忍者对尾组 |
| **头组得分** |
| 36 | `h_card_chips` | int | 牌面筹码(3张之和) |
| 37 | `h_hand_chips` | int | 牌型筹码 base |
| 38 | `h_chips_total` | int | 总筹码(card+hand+ench+ninja) |
| 39 | `h_hand_mult` | int | 牌型倍率 base |
| 40 | `h_mult_total` | int | 总倍率(hand+ninja) |
| 41 | `h_chips_mul` | int | chips_total × mult_total = 原始分 |
| 42 | `h_score` | int | 最终组分(= ×product of 组内x_stack) |
| **中组得分**（同上模式 43-47） |
| **尾组得分**（同上模式 48-52） |
| 53 | `total_raw` | int | 头+中+尾 |
| 54 | `col_x_prod` | int | 列×mult 累乘积 |
| 55 | `xi_x_prod` | int | 喜×mult 累乘积 |
| 56 | `final_score` | int | 最终得分 |

## 三、核心展开示例

### 📌 n_001 手里剑 T1 — 全散牌基准

```
头:(♣5=5 + ♠3=3 + ♥9=9) = 17chips + 5(散牌) = 22, ×1mult = 22
中:(♦6=6 + ♣4=4 + ♥10=10) = 20chips + 5(散牌) = 25, ×1 = 25
尾:(♣2=2 + ♦A=11 + ♥8=8) = 21chips + 5(散牌) = 26, ×1 = 26
                                    基准: 22+25+26 = 73

列0:♣5♦6♣2 → 2,5,6 散牌×1
列1:♠3♣4♦A → 3,4,14 散牌×1
列2:♥9♥10♥8 → 8,9,10 ♥同花顺×16

基准: 73 × 16 = 1,168
n_001(+10chips全组): 32+35+36=103 × 16 = 1,648
Δ = (10+10+10) × 16 = 480 ✓
```

### 📌 n_g01 虎头 T1 — 头散牌触发 +5 mult

```
头: ♠9=9 + ♥10=10 + ♣Q=10 = 29chips + 5(散牌) = 34, × (1+5=6) = 204   ← +5mult生效
中: ♠8=8 + ♥8=8 + ♣J=10 = 26chips + 10(对子) = 36, × 2 = 72
尾: ♦J=10 + ♦Q=10 + ♦K=10 = 30chips + 50(同花顺) = 80, ×5 = 400
                                    基准: 34×1 + 36×2 + 80×5 = 34+72+400 = 506
                                    有虎头: 34×6 + 72 + 400 = 204+72+400 = 676
Δ = 34×(6-1) = 170 ✓
```

### 📌 n_g02 龙尾 T2 — 尾豹子触发×2 + 列顺子×4

```
头: 同n_g01 T1: 21×1=21
中: 同: 26×1=26
尾: ♥J=10 + ♠J=10 + ♦J=10 = 30chips + 100(豹子) = 130, ×8 = 1,040
    → n_g02: 130 ×8 ×2 = 2,080    ← ×2生效

基准total_raw = 21+26+1,040 = 1,087
基准final = 1,087 ×4(列) = 4,348

有龙尾total_raw = 21+26+2,080 = 2,127
有龙尾final = 2,127 ×4 = 8,508

Δ = 1,040 ×4 = 4,160 ✓
```
