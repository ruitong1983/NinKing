## Blind 结构

- **8 Antes**，每个 Ante = **3 Blinds**（Small / Big / Boss）
- 每个 Blind 结束后进入商店，Boss Blind 后进入 Ante 结束商店（奖励更多）
- Small/Big Blind 商店: 2 忍者牌 + 1 附魔 + 1 星图
- Boss Blind 商店: 3 忍者牌 + 2 附魔 + 2 星图

### 难度曲线设计意图 (v5.0)

| 阶段 | Ante | 设计意图 |
|------|------|---------|
| 入门 | 1-3 | 基础牌型可过，新手 2-3 次出牌过关。列加法化后列贡献温和(1-2列对子+60~120分)，不出现"碰到对子列就秒 Boss" |

## Boss Blind (10)

每个 Boss 含一条规则变动。Boss 在前一个 Big Blind 时随机选定（Ante 对应池中随机抽取）。

### Boss 分配

| Ante | Boss 池 | 说明 |
|------|---------|------|
| 2 | 断尾 / 无头 | 单一组失效，引入 Boss 概念 |
| 3 | 独柱 / 铁链 | 加大压力 |
| 4 | 反目 / 封印师 / 散牌王 | 规则扭曲 |
| 5 | 饿鬼 / 散牌王 | 经济/牌型限制 |
| 6 | 喜之克星 | 针对喜流派 |
| 7 | 喜之克星 | 针对喜流派 |
| 8 | 终焉 | 终极考验 |

### Boss 完整列表

| # | Boss | 效果 | Ante | 状态 |

| 1 | 断尾 | 滅不计分 | 2 | ✅ |
| 2 | 无头 | 影不计分 | 2 | ✅ |

### Boss 设计原则

| 原则 | 说明 |
|------|------|
| 可应对 | 每个 Boss 都有应对策略（调整排列 / 换牌 / 忍者牌组合） |
| 不叠加惩罚 | 终焉不叠加多惩罚——单一规则变更已足够难 |
| 非 RNG 关键效果 | 封印师从随机组改为最低分组——玩家可通过排列控制 |
| 前移温和 Boss | 散牌王移到 Ante 4-5，留够时间给玩家调整构筑 |

### AI 适配

当 Boss 改变约束/计分规则时，AI 自动排列器（`auto_arranger.gd`）必须对应适配：

| Boss | AI 行为 |
|------|---------|
| 断尾/无头/独柱 | 在被跳过组中不再优先排列高牌力 |
| 反目 | 约束过滤反转：要求 head ≥ mid ≥ tail（影 ≥ 瞬 ≥ 滅） |
| 散牌王 | 排列策略改为最大化牌面筹码 + 星图等级 |
| 封印师 | 尝试让三组忍気均匀分布（降低最低忍気的方差） |
