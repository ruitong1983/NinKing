# 忍者牌完整卡面 — 即梦提示词清单 v5.0

> **方案：** Balatro 风纯插画卡面，无文字，每张靠独特视觉钩子辨识
> **背景方案：** 极暗墨色背景 — 视觉近黑但黑描边可辨（非纯黑 #000）
> **类型：** 整卡插画，非透明 PNG，无需抠图

---

## 统一出图参数（全部 44 张共用）

| 参数 | 值 |
|------|-----|
| **模型** | 图片4.0 |
| **画质** | 4K |
| **尺寸** | 1000×1400px（5:7 竖版） |
| **生成数量** | 各 1 张 |
| **背景** | 极暗墨色（约 #0a0a10~#111118，非纯黑 #000） |
| **透明** | 否 — 整卡插画，无需透明通道 |
| **文字** | 画面内不得出现任何文字/字母/数字/符号 |

### 统一反向提示词（全部 44 张共用）

```
文字，字母，数字，签名，水印，logo，标题，横幅，边框，UI元素，
低质量，模糊，扭曲，变形，崩坏，多余手指，比例失调，
真实照片，写实，3D渲染，照片级，真人，
白色背景，彩色背景，渐变背景，明亮背景，灰色背景，纯黑背景，
线稿，草图，未完成，
横版构图
```

### 统一正向风格词（每张末尾追加）

```
少年漫画风，粗黑描边，cel-shading，网点阴影，极暗墨色背景
```
