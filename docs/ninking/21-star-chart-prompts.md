# 南斗六星·星图卡 — 即梦提示词清单 v1.0

> **方案：** 南斗六星主题纯插画卡面，无文字，每张以天文星象+对应牌型视觉元素区分
> **背景方案：** 深蓝紫夜幕（#080818~#0f0f28），点缀疏密星辰
> **类型：** 整卡插画，非透明 PNG，无需抠图
> **输出目录：** `assets/images/items/star_charts/`（新建）

---

## ⚡ 即梦操作流程（每次出图）

```
每张卡出图步骤：
1. 设置右侧参数：图片4.0 / 4K / 1000×1400px
2. 在反向提示词框粘贴「统一反向提示词」（见下方）
3. 在正向提示词框粘贴该卡的「正向」内容（含尺寸和风格词）
4. 点击生成
```

> 正向提示词已包含完整尺寸和风格词，直接复制整段粘贴即可。
