# NinKing 商店 UI 设计方案 v2

> 参考：Balatro 小丑牌商店界面 | 适配：NinKing 扑克牌型计分闯关
> Figma 设计稿：`docs/ninking/figma-shop-v2.png`

---

## 一、设计目标

精确复刻 Balatro 商店的核心视觉语言：

- **卡片即主角** — 商品以大尺寸卡片展示，含艺术插画区、名称牌、效果描述
- **暗底 + 金色点缀** — 深绿桌面底 + 金色标题/边框/价格
- **卡框质感和光影** — 圆角卡框 + 内边框 + 投影 + 稀有度色彩区分
- **适配项目配色**：`#0F2822` 面板 / `#D4A843` 金色 / `#E04040` 红色 / `#F0D060` 金币 / `#8B5CF6` 紫(能力) / `#4080D0` 蓝(道具)

## 五、交互流程

### 5.1 进入商店

```
过关弹窗 → 点击 [进入商店] → Tween 过渡:
  1. 遮罩淡入 0.3s
  2. 面板从下方滑入 (y: +200 → 0, 0.4s, ease OutBack)
  3. 金币数跳动到实际值 0.5s
  4. 卡片依次弹出 (每张延迟 0.08s, scale 0→1, ease OutBack)
```

### 5.2 购买

```
点击 [购买] →
  ✓ 成功: 金币递减动画 + 卡片金色闪烁 + 按钮变灰 "已购买"
  ✗ 金币不足: 按钮红色闪烁 + Toast "金币不足!"
  ✗ 槽位已满: Toast "能力牌槽位已满 (5/5)!"
```

### 5.3 Reroll (刷新商店)

```
点击 🔄$5 →
  - 消耗 5 金币
  - 所有卡片翻转动画 0.3s
  - 重新随机抽取商品
```

### 5.4 继续闯关

```
面板向下滑出 0.3s → 遮罩淡出 → 进入下一关 LevelIntro
```
