# NinKing 游戏设计文档 (v1.0 废弃版)

## 概述

NinKing 是一款融合**比鸡（9张牌三墩）**与**小丑牌（Balatro）Roguelike 计分闯关**的扑克游戏。玩家将9张手牌排成头/中/尾三墩（必须满足头≤中≤尾牌力升序约束），三组同时计分，累计得分达到关卡目标即可过关。

- **类型**: 单人 Roguelike 卡牌计分
- **平台**: Windows (Godot 4.6.2)
- **视觉**: 复古像素风 + CRT 滤镜
- **规模**: 紧凑 MVP — 8 关 / 30 能力牌 / 10 道具卡
- **完整设计**: 参见 `docs/ninking/06-complete-redesign.md`

## 核心规则

### 基本参数

| 参数 | 值 |
|---|---|
| 手牌数量 | **9 张**（排列为头墩[0-2] / 中墩[3-5] / 尾墩[6-8]） |
| 每关出牌次数 | **3 次**（每次9张全出，不保留手牌） |
| 每关换牌次数 | **2 次**（每次丢弃 1~3 张后补抽，自动重排） |
| 计分公式 | (筹码) × (倍率)，三组合并计算 |
| 牌型 | **6 种 3 张牌型**：散牌 < 对子 < 顺子 < 同花 < 同花顺 < 豹子 |
| 约束 | **头墩牌力 ≤ 中墩牌力 ≤ 尾墩牌力**（不满足不可出牌） |
| 起始牌组 | 标准 52 张 |
| 能力牌槽位 | 5 个 |

### 回合流程

```
发 9 张 → 自动排墩(头≤中≤尾) → 玩家可选:
  ├── [交换] 点击两张牌互换位置调整排列
  ├── [换牌] 选 1~3 张丢弃 → 补抽 → 自动重排 (每关 2 次)
  └── [确认出牌] 约束满足 → 三组计分 → 检查过关 (每关 3 次)
        │
        ├── 达标 → 过关 (进入商店)
        └── 3 次出牌用完未达标 → 失败

## 计分系统

### chips × mult 双因子模型（三组合并）

```
最终得分 = (9张牌面筹码 + 三组牌型筹码之和 + 能力牌筹码) × (三组牌型倍率之和 + 能力牌倍率)
```

### 牌面筹码值

| 牌面 | 筹码 |
|---|---|
| A | 11 |
| K, Q, J | 10 |
| 10 ~ 2 | 面值 (2~10) |

### 3张牌型基础值

| 牌型 | 筹码 | 倍率 |
|---|---|---|
| 豹子 (Three of a Kind) | 40 | 5 |
| 同花顺 (Straight Flush) | 35 | 4 |
| 同花 (Flush) | 20 | 3 |
| 顺子 (Straight) | 20 | 3 |
| 对子 (One Pair) | 10 | 2 |
| 散牌 (High Card) | 5 | 1 |

### 喜系统（9张牌全局模式）

每次出牌后检测，触发的喜加成乘入总倍率：
- **全黑**：9张全 ♠/♣ → ×1.5
- **全红**：9张全 ♥/♦ → ×1.5
- **四张**：手中有4张同点数 → +50 chips
- **三清**：三组全是同花 → ×1.3

### 小丑牌效果类型

- **add_chips**: 增加筹码（加法）
- **add_mult**: 增加倍率（加法）
- **x_mult**: 倍率翻倍（乘法）
- **condition**: 条件触发（特定牌型 / 含 Ace / 含人牌 等）

## 道具系统

### 小丑牌 (Jokers)

- 数量: 30 张
- 效果: 持续被动 buff，每次计分自动生效
- 槽位上限: 5 个
- 获取方式: 商店购买
- 分类:
  - 通用加成 (4 张) — 无条件加筹码/倍率
  - 牌型触发 (9 张) — 特定牌型时触发
  - 人牌/Ace 触发 (4 张) — 含特定牌面时触发
  - 倍率增强 (3 张) — 高倍率加成
  - 混合效果 (5 张) — 筹码+倍率组合
  - 组合加成 (5 张) — 特定牌型的高额加成

### 道具卡 (Items)

- 数量: 10 种
- 效果: 一次性使用，仅计分前可用
- 获取方式: 商店购买
- 使用: 换牌前选择使用，对当次计分生效

## 操作交互

1. **主菜单**: 点击「开始游戏」→ 进入第 1 关
2. **选牌**: 点击手牌选中（高亮红色），可多选 1~5 张
3. **操作按钮**: 选中卡牌后在手牌上方弹出「出牌」「换牌」
   - 选满 5 张 → 出牌可用，点击出牌直接计分
   - 选 1~5 张 → 换牌可用，点击换牌丢弃选中卡牌并补抽
   - 取消全部选中 → 按钮隐藏
4. **计分**: 出牌后自动计分，显示牌型和得分
5. **过关**: 弹窗显示过关节果 → 点击进入商店
6. **商店**: 购买小丑牌/道具卡 → 点击「继续闯关」进入下一关
7. **牌库**: 点击换牌区下方「🎴 牌库」按钮 → 弹出牌库查看器，展示剩余卡牌（按花色/点数排序），点击遮罩或 ✕ 关闭
8. **失败**: 回到主菜单
