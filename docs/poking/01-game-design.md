# Poking 游戏设计文档

## 概述

Poking 是一款小丑牌（Balatro）风格的扑克牌型计分闯关游戏。玩家通过换牌机制打出扑克牌型，累计得分达到关卡目标即可过关。

- **类型**: 单人 Roguelike 卡牌计分
- **平台**: Windows (Godot 4.6.2)
- **视觉**: 复古像素风 + CRT 滤镜
- **规模**: 紧凑 MVP — 8 关 / 30 小丑牌 / 10 道具卡

---

## 核心规则

### 基本参数

| 参数 | 值 |
|---|---|
| 手牌数量 | 8 张 |
| 每关换牌次数 | 固定 5 次 |
| 计分公式 | (筹码) × (倍率) |
| 牌型 | 标准 10 种扑克牌型 |
| 起始牌组 | 标准 52 张 (2 种初始牌组，MVP 参数相同) |
| 小丑牌槽位 | 5 个 |

### 回合流程

```
手牌 8 张开局
  │
  └─→ 玩家选择换 N 张牌 (1 ≤ N ≤ 5)
        ├─ N = 5 → 这 5 张直接打出计分
        ├─ N < 5 → 这 N 张作底 + 从剩余手牌补 (5−N) 张 → 5 张计分
        │
        ├─ 打出 5 张 → 弃牌堆
        ├─ 从牌库抽 5 张补满手牌至 8 张
        ├─ 牌库抽空 → 弃牌堆重洗
        └─ 累计得分 ≥ 目标分数 → 过关
```

### 失败条件

- 5 次换牌机会用完，累计得分未达到目标分数 → **从头开始**（Roguelike 永久死亡）

## 计分系统

### chips × mult 双因子模型

```
最终得分 = (牌面筹码 + 牌型筹码 + 小丑牌筹码) × (牌型倍率 + 小丑牌倍率)
```

### 牌面筹码值

| 牌面 | 筹码 |
|---|---|
| A | 11 |
| K, Q, J | 10 |
| 10 ~ 2 | 面值 (2~10) |

### 牌型基础值

| 牌型 | 筹码 | 倍率 |
|---|---|---|
| 皇家同花顺 | 100 | 8 |
| 同花顺 | 80 | 8 |
| 四条 | 60 | 7 |
| 葫芦 | 40 | 4 |
| 同花 | 35 | 4 |
| 顺子 | 30 | 4 |
| 三条 | 30 | 3 |
| 两对 | 20 | 2 |
| 一对 | 10 | 2 |
| 高牌 | 5 | 1 |

### 小丑牌效果类型

- **add_chips**: 增加筹码（加法）
- **add_mult**: 增加倍率（加法）
- **x_mult**: 倍率翻倍（乘法）
- **condition**: 条件触发（特定牌型 / 含 Ace / 含人牌 等）

## 道具系统

### 小丑牌 (Jokers)

- 数量: 30 张
- 效果: 持续被动 buff，每次计分自动生效
- 槽位上限: 5 个
- 获取方式: 商店购买
- 分类:
  - 通用加成 (4 张) — 无条件加筹码/倍率
  - 牌型触发 (9 张) — 特定牌型时触发
  - 人牌/Ace 触发 (4 张) — 含特定牌面时触发
  - 倍率增强 (3 张) — 高倍率加成
  - 混合效果 (5 张) — 筹码+倍率组合
  - 组合加成 (5 张) — 特定牌型的高额加成

### 道具卡 (Items)

- 数量: 10 种
- 效果: 一次性使用，仅计分前可用
- 获取方式: 商店购买
- 使用: 换牌前选择使用，对当次计分生效

## 操作交互

1. **主菜单**: 点击「开始游戏」→ 进入第 1 关
2. **选牌**: 点击手牌选中（高亮红色），可多选 1~5 张
3. **操作按钮**: 选中卡牌后在手牌上方弹出「出牌」「换牌」
   - 选满 5 张 → 出牌可用，点击出牌直接计分
   - 选 1~5 张 → 换牌可用，点击换牌丢弃选中卡牌并补抽
   - 取消全部选中 → 按钮隐藏
4. **计分**: 出牌后自动计分，显示牌型和得分
5. **过关**: 弹窗显示过关节果 → 点击进入商店
6. **商店**: 购买小丑牌/道具卡 → 点击「继续闯关」进入下一关
7. **失败**: 回到主菜单
